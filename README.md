# Worm-on-the-knee

 Простерькая игра змейка для браузера написанная на Javascript


Демонстрационная ссылка: https://yurkaronin.github.io/Worm-on-the-knee/


Доработки

давай добавим в игру статистику прохождения.
в начале игры игрок должен выбрать ник и нажать кнопку начать игру.
при этом нужно проверять не занято ли это имя игрока.
Данные игроков хранятся в отдельном файле, который нужно создать.
Хранить нужно дату игровой сессии, время по Москве, имя игрока, его лучший результат прохождения игры (по количеству набранных очков).

в конце игры помимо сообщения и кнопки "Играть еще" выводить рейтинговую таблицу со списком TOP10 игроков у которые набрали наибольшее количество очков.
данные с ником игрока сохранять в браузере.
при загрузке игры в браузере проверять играл ли раньше пользователь в игру.
Если играл - поздороваться с игроком: 'С возвращением, 'Имя игрока'"
